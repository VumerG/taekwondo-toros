## 📋 Qué Falta: Los Pasos del Usuario

Tu aplicación está 100% lista. Solo necesitas hacer esto:

### Paso 1: Crear Proyecto Firebase (2 minutos)
```
1. Ve a: https://console.firebase.google.com/
2. Crea proyecto "taekwondo-toros"
3. Esperar a que se cree
```

### Paso 2: Registrar App Web (2 minutos)
```
1. Click en </> (Web)
2. Copia tus credenciales
3. Pégalas en .env.local
```

### Paso 3: Configurar Firestore (2 minutos)
```
1. Cloud Firestore → Crear base de datos
2. Modo: "Iniciar en modo de prueba"
```

### Paso 4: Configurar Authentication (2 minutos)
```
1. Authentication → Correo electrónico/Contraseña
2. Crear 3 usuarios de prueba
```

### Paso 5: Instalar Firebase CLI (2 minutos)
```bash
npm install -g firebase-tools
firebase login
```

### Paso 6: Desplegar (2 minutos)
```bash
npm run build
firebase deploy
```

**Total: ~15 minutos** ✅
